# Advanced Multi-AS BGP Network using Cisco CML

## Overview

This project demonstrates the design and implementation of an advanced multi-autonomous system network using Cisco Modeling Labs (CML).

The network contains three autonomous systems connected using eBGP and iBGP. Several advanced BGP techniques were implemented to improve scalability, reliability, and traffic engineering.

## Features

- eBGP
- iBGP
- Route Reflectors
- BGP Local Preference
- Default Route Advertisement
- Route Summarisation
- BGP Communities (no-export)
- Static Routing
- Route Filtering
- Traffic Engineering

## Technologies

- Cisco Modeling Labs (CML)
- Cisco IOS
- BGP
- OSPF
- Static Routing

## Learning Outcomes

This project demonstrates practical experience with enterprise and service-provider BGP technologies including routing policy implementation, route filtering, redundancy, and network troubleshooting.
